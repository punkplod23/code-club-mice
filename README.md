## ðŸ§  The MICE Framework
When social engineers target an individual, they look for one of four "bugs" in human nature:

| Letter | Motivator | Description |
| :--- | :--- | :--- |
| **M** | **Money** | Targets financial stress, greed, or the need for quick cash. |
| **I** | **Ideology** | Targets beliefs. Persuading someone that "doing the wrong thing" is for a "greater good." |
| **C** | **Coercion** | Using pressure, threats, or blackmail to force someone's hand. |
| **E** | **Ego** | Appealing to vanity, the desire for recognition, or the need to feel smart. |
